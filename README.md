


# SaaS Campaign Dashboard

## Overview
A React + Tailwind SaaS dashboard for managing marketing campaigns.

## Features
- Campaign Management Table
- Search and Status Filters
- Sorting and Pagination
- Bulk Campaign Selection
- Campaign Detail Page
- Editable Overview Form
- Asset Upload Simulation
- Performance Charts

## Tech Stack
- React
- TailwindCSS
- React Router
- Recharts

## Installation
npm install
npm run dev
